--- Conclusion ---
The Phase 2b generation is runtime-observed (terminal output + local event chain),
bus-backed for delegated inference, but NOT projection-backed. A projection row
requires the server-side runtime path, which the contractor seat cannot trigger
(no raw bus publish / runtime trigger per plan rules). Projection confirmation
needs operator-side capture. This is a scope/architecture boundary, not a defect.
